Definición general

Aplicación SPA que lleva registros del clima y reporta los diversos cambios climatológicos en los sectores agrícolas de las provincias de Entre Ríos, Córdoba y Buenos Aires.

Especificaciones

El sistema provee de manera fiel y en tiempo real los datos del clima de las zonas agrícolas de 3 provincias (Entre Ríos, Córdoba, Buenos Aires) y lleva registro de ello.

  • Vistas del mapeo del clima.
  • Reportes con manejo de filtros para generar registro de historiales.
  • Vistas de alertas en casos de posibles tormentas.

Encuentra el clima de cualquier ciudad por su nombre, código de país o estado y zip/código postal.

  • Mapa

    Mapa interactivo con diferentes capas climáticas.

  • Alertas

    Aviso de posibles tormentas según el volumen de precipitación en las ciudades de 3 provincias (Buenos Aires, Córdoba, Entre Ríos).

  • Reportes

    Registro histórico de los datos del clima.

  • Buscador

    Encuentra los datos meteorológicos de mas de 200.000 ciudades en el mundo.

Herramientas

Mediante el uso de llamadas API hacia OpenWeatherMap , se reúnen los datos e información suficientes para crear nuestra REST API con el uso del framework Loopback.

Angular

Plataforma de código abierto para construir aplicaciones web de una sola página (SPA).

Loopback 4

Framework de Node.js, permite crear API y microservicios dese bases de datos a servicios SOAP y REST.

MongoDB

Base de datos de documentos que ofrece una gran escalabilidad y flexibilidad, y un modelo de consultas e indexación avanzado.

PushingBox

Es una nube que puede enviar notificaciones en tiempo real basadas en llamadas API. A partir de una misma solicitud, puedes enviar varias notificaciones como un Push, un Tweet, un Email.

Git

Es un sistema de control de versiones distribuido, gratuito y de código abierto diseñado para manejar todo con rapidez y eficiencia, desde proyectos pequeños hasta proyectos muy grandes.

Leaflet

Leaflet es una biblioteca JavaScript de código abierto que permite crear mapas interactivos en la web.
